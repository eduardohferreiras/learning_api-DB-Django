Welcome to my social network API!

In this API you can:

  • Get all profiles (/profiles GET);
  • Get a specific profile by id (/profiles/{id:int} GET);
  • Create a new profile (/profiles POST);
  • Hide or unhide an existing profile from the connection recomendation algorithm (/profiles/{id:int} PATCH);
  • Get all connections (/connections GET);
  • Create a new connection (/connections POST);
  • Get a connection recomendation for an existing profile (/connection POST);
